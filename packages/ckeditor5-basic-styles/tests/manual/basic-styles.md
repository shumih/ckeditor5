## Basic styles

1. The data should be loaded with:
  * italic _"This"_,
  * bold **"editor"**,
  * underline "instance",
  * strikethrough ~~"is"~~,
  * code `"an"`,
  * subscript X<sub>1</sub>,
  * superscript X<sup>2</sup>.
2. Test the bold, italic, strikethrough, underline, code, subscript and superscript features live.
